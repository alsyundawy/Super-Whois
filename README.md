# Super Whois
域名、IP WHOIS 搜索系统 
支持IDN域名

# 使用方法：
1.在下方输入框中输入要查询的域名或 IP 地址，然后选择查询类型。  
2.点击提交按钮进行查询。  
3.查询结果将会显示在下方。
